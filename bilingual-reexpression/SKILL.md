# 英→繁中（台灣）語意重述指引

這不是翻譯——是讀懂英文要說什麼，然後用**台灣人說話的方式**重新把同一件事寫出來。正確做法：讀懂原文意思後把英文蓋住，用繁體中文從頭說一遍。目標：成品完全讀不出是從英文來的。此 skill 專注於**書面語**（文件、文章、技術文件、正式文書）。

---

## 關鍵差異：為什麼直譯行不通

| 差異類型 | 中文規則 |
|---------|---------|
| 字面意思 ≠ 實際意思 | Idioms、俗語、片語動詞一律解碼語義後重述，絕不直譯字面 |
| 形合 vs. 意合 | 刪除多餘連接詞，靠語序銜接邏輯關係 |
| 顯性 vs. 隱性 | 省略上下文可推斷的主語、代詞、時態標記 |
| 尾重 vs. 頭重 | 重組語序：背景／原因在前，結論／動作在後 |
| 否定提升（Neg-raising）| "I don't think X" → 否定移入從句：「我認為 X 不…」「我覺得 X 大概不…」 |
| 話題優先 vs. 主語優先 | 英文抽象名詞主語改用話題前置：「就 X 而言」「X，…」 |

詳細說明與對照範例見 `references/en-to-zh.md` §16。

## 工作流程

### 第一步：讀透原文
- 用一句大白話說出這段英文在講什麼
- 標記所有 idioms 與非字面表達，解碼成實際語義（不帶字面進下一步）
- 若原文有歧義，先釐清再動筆（見 `references/en-to-zh.md` §14）

### 第二步：拆解句構
- 找出每句 SVO 主幹；一個長句可能有多個主幹
- 超過一個 which/that/because/although/-ing 分詞的句子幾乎一定要拆
- 重排資訊順序：時間／背景 → 原因／條件 → 動作 → 結果
- 詳細決策樹見 `references/en-to-zh.md` §11

### 第三步：用台灣中文從頭說
- 蓋住原文，按第二步順序直接用中文寫
- 短句流動，靠語義銜接，不靠連接詞硬串
- 用台灣慣用詞（見下方對照表），不用大陸用語

### 第四步：對齊語氣與文體
- 比對 must/should/may/might 的力度，確保中文語氣相符
- 完整情態映射表見 `references/en-to-zh.md` §13；文體要點見 §8

### 第五步：三層修訂
1. **語意正確**：核心訊息無遺漏、非字面表達未誤直譯、語氣力度一致
2. **中文自然**：用下方診斷表掃除翻譯腔；低聲唸一遍，停頓處即為問題點
3. **用語一致**：台灣用語全篇統一；同一概念只用一個中文詞（術語策略見 §12）

**停止條件**：三層通過、且成品讀起來不像翻譯過來的，即完成。

## 翻譯腔診斷框架

| # | 症狀 | 怎麼修 |
|---|------|--------|
| 1 | **字面直譯非字面表達** | 從原文語義重述，不轉字面 |
| 2 | **尾重句型未重組** | 修飾前移，改為「背景→動作→結果」|
| 3 | **「的」字連環**（同句三個以上）| 拆成短句，或以動詞取代名詞化定語 |
| 4 | **被動句濫用** | 還原主動語態，或改用「獲」「經」「受」|
| 5 | **代詞殘留** | 省略上下文可推斷的代詞 |
| 6 | **連接詞堆疊** | 靠語序銜接，轉折不明顯時才保留 |
| 7 | **空泛動詞**（「進行」「作出」＋名詞）| 直接換成具體動詞 |
| 8 | **名詞化堆砌**（「…的改進/實施」）| 把名詞還原為動詞 |
| 9 | **陸式用語混入** | 換成台灣慣用詞（見下方對照表）|
| 10 | **否定位置錯置**（「我不認為…」）| 否定移入從句（見上方「否定提升」規則）|

每個症狀的展開說明與多組範例見 `references/en-to-zh.md` §10。

## 台灣用語 vs. 中國大陸用語

| 英文 | 台灣 ✓ | 大陸 ✗ |
|------|--------|--------|
| software / hardware | 軟體 / 硬體 | 軟件 / 硬件 |
| information / data | 資訊 / 資料 | 信息 / 數據 |
| network / program | 網路 / 程式 | 網絡 / 程序 |
| video / server | 影片 / 伺服器 | 視頻 / 服務器 |
| default / bandwidth | 預設 / 頻寬 | 默認 / 帶寬 |
| quality / support | 品質 / 支援 | 質量 / 支持 |
| mobile / blog | 行動 / 部落格 | 移動 / 博客 |
| memory (RAM) / mouse | 記憶體 / 滑鼠 | 內存 / 鼠標 |

完整詞彙對照見 `references/en-to-zh.md` §9。

## 邊界說明

此 skill 只處理**英文→繁體中文（台灣）的語意重述**。以下不在範圍內：
- 中文→英文方向
- 特定領域術語表（法律、醫學、金融等）——建議使用者依需求自建術語表
- 口語、對話、UI 文案等非書面語體裁
- 逐字逐句的直譯（如果你要的是直譯，這個 skill 不適用）
